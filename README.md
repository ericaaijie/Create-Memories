## Connect Google Sheets

1. Upload `Create_Memories_Google_Sheets_CMS.xlsx` to Google Drive and open it as Google Sheets.
2. Share it as **Anyone with the link — Viewer** or publish it to the web.
3. Copy the spreadsheet ID from the URL.
4. Paste the ID into `assets/js/config.js`.
5. Upload the full project to GitHub Pages.

Do not rename these tabs: Config, Route, Itinerary, Activities, Hotels, Food, Attractions.

If Sheets cannot load, the site safely falls back to the corrected built-in itinerary.
